## Usage
```
Usage: bluegone [OPTIONS] [COMMAND]

Commands:
  list-presets
  set
  daemon
  help          Print this message or the help of the given subcommand(s)

Options:
  -b, --backend <BACKEND>  Backend to use (X11 or TTY)
  -h, --help               Print help
  -V, --version            Print version
```
